# system-of-guielle
⚙️ Modélisation et simulation numérique du Système de Guielle — un mécanisme  élastique à n ressorts entraînant une roue en rotation continue.  Inclut la dérivation mathématique complète, l'analyse de la triple condition  sur g(t), et un simulateur interactif Streamlit avec intégration RK4.
